# WatchCom

AI-powered movie recommendation platform focused on understanding user taste through ratings, Taste DNA analytics, and explainable recommendations.

## Overview

WatchCom is designed around a simple idea:

**Rate Movies → Build Taste DNA → Get Personalized Recommendations**

Unlike traditional movie catalogs or streaming platforms, WatchCom focuses on understanding user preferences and explaining why recommendations are generated.

## Features

* Taste DNA Analytics
* Personalized Recommendations
* Recommendation Explanations
* Profile Completion Tracking
* Recommendation Confidence Metrics
* Engine Status Dashboard
* Modern Responsive UI
* Dark Luxury Design System
